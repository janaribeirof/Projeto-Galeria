## ♻ How Contribute or Setup
<hr>

```bash
# Clone the project
  $ git clone https://github.com/janaribeirof/ProjectGallery.git
````
```bash
  # Enter directory
  $ cd ProjectGallery
```
```bash
  # Install the dependencies, if use npm
  $ npm install
```
<p align="center">or</p>

```bash
  # Install the dependencies, if use yarn
  $ yarn
```
